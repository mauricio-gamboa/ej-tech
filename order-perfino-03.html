Licensing FAQ

I want to monitor N VMs. How many licenses do I need?

You need one product key and N-1 additional VM keys.

What happens if I try to monitor more VMs than I have licenses for?

The VMs that connect after all licenses have been exhausted will be shown in the VMs view in perfino, but without any data and a name prefix of [UNLICENSED]. As soon as you add more license keys in the perfino UI, monitoring will start for the newly licensed VMs.

How can I try out my monitoring configuration?

You can make additional installations of the perfino server for staging and development use. Those installations must be purely internal and not be used for business purposes.

Are there any platform restrictions?

No, perfino can be used on any platform where a HotSpot VM is available.

Can I redistribute the monitoring agent?

Yes, you can redistribute the perfino monitoring agent, so you can monitor VMs that are running in third party locations. The perfino API is also redistributable.

What upgrades will I get?

Minor upgrades are always free. A major upgrade is free of charge when it is released during the support period (60 days for a standard license, 1 year for a support package).
